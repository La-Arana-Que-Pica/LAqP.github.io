Resumen

Este Option File reúne varias ligas y equipos de Sudamérica y CONCACAF para PES 2020, con todo el contenido actualizado a esta temporada. Está pensado para quienes quieren jugar torneos sudamericanos con planteles y clubes al día.

Contenido del Option File

  • Liga Argentina.
  • Liga de Brasil.
  • Liga de Chile.
  • Liga Uruguaya.
  • Equipos de CONCACAF.
  • Equipos de otros países latinoamericanos para jugar la Copa Libertadores.

Todo el contenido está actualizado a la temporada 2025, según el trabajo del creador.

Caras y estadísticas

El Option File incluye caras y estadísticas trabajadas por Bruno Herrera para que los jugadores y equipos de estas ligas se sientan más fieles a la realidad.

Instalación en PS4/PS5 (PES 2020)

  1. Entrar al video original y seguir el link de descarga que está en el comentario fijado.
  2. Descargar el archivo del creador original.
  3. Extraer el contenido del archivo descargado.
  4. Copiar la carpeta WEPES en una memoria USB (formato FAT32).
  5. En PES 2020 ingresar a: Editar → Importar/Exportar → Importar equipos/competiciones.
  6. Seleccionar todos los archivos y activar las casillas recomendadas (aplicar datos a todos los equipos / sobrescribir duplicados, según indique el creador).
  7. Guardar los cambios antes de salir.
